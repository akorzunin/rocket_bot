# rocket_bot

## Install

- `.env` file example:

    ```
        BOT_USERNAME=123
        BOT_PASSWORD=123
        ROCKETCHAT_SERVER_URL=http://192.168.1.125:3000/
        HOST=0.0.0.0
        PORT=8002
        CHAT_NAME=GENERAL
        BOT_PREFIX=.
    ```
- Docker

    ```
        docker compose -f "docker-compose.yml" up -d --build
    ```
